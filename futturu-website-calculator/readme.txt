Calculadora interativa para orçamentos de websites com personalização completa de cores e fontes.

== Description ==

A **Calculadora de Criação de Website Futturu** é um plugin WordPress completo que permite aos seus potenciais clientes calcularem, de forma interativa e transparente, um orçamento estimado para a criação de um website.

= Funcionalidades Principais =

* **Calculadora Interativa**: 5 campos dinâmicos (Tipo de Website, Complexidade, Número de Páginas, Extras e Hospedagem)
* **Cálculo em Tempo Real**: Atualização automática do valor conforme o usuário seleciona as opções
* **Painel Administrativo Completo**: Configure todos os valores, multiplicadores e opções
* **Personalização Visual Avançada**:
  - Escolha de cores (primária, secundária, destaque, fundo, texto)
  - Seleção de fontes (8 opções profissionais)
  - Arredondamento de bordas ajustável
  - Estilos de botão (sólido, gradiente, contorno)
* **Formulário de Contato Integrado**: Captura leads com todos os detalhes do projeto
* **Envio de E-mail Automático**: Envia orçamento detalhado para suporte@futturu.com.br
* **Totalmente Responsivo**: Funciona perfeitamente em dispositivos móveis e desktop
* **Compatível com Qualquer Tema**: Usa shortcode `[futturu_calc]`
* **Open Source**: Sem dependências externas pesadas

= Campos da Calculadora =

1. **Tipo de Website**: Institucional, Blog, Landing Page, Hotsite
2. **Nível de Complexidade**: Baixa, Média, Alta (com multiplicadores percentuais)
3. **Número de Páginas**: Valor configurável por página adicional
4. **Extras e Aplicações**: 
   - Formulário de Contato Avançado
   - Integração com API Externa
   - Catálogo de Produtos Estático
   - Blog Integrado
   - Área de Membros Simples
   - Chat Online
   - Integração com Redes Sociais
   - SEO Básico
   - Google Analytics
   - Multi-idioma
5. **Plano de Hospedagem Cloud**: Básico, Profissional, Empresarial (valor mensal separado)

= Personalização de Aparência =

O plugin inclui uma seção completa de personalização visual no painel administrativo:

* **Cor Primária**: Cor principal dos elementos e botões
* **Cor Secundária**: Cor de hover e detalhes secundários
* **Cor de Destaque**: Cor para detalhes especiais e resultados
* **Cor de Fundo**: Fundo da calculadora
* **Cor do Texto**: Cor do texto principal
* **Fonte**: 8 opções de famílias tipográficas profissionais
* **Arredondamento de Bordas**: Ajuste de 0 a 50px
* **Estilo do Botão**: Sólido, Gradiente ou Contorno

= Como Usar =

1. Instale e ative o plugin
2. Acesse **Configurações > Calculadora Futturu** no menu admin
3. Configure os valores de cada opção conforme sua tabela de preços
4. Personalize as cores e fontes para combinar com sua marca
5. Use o shortcode `[futturu_calc]` em qualquer página ou post

= Requisitos Técnicos =

* WordPress 5.0 ou superior
* PHP 7.4 ou superior
* jQuery (incluído no WordPress)

== Installation ==

= Instalação Manual =

1. Faça o upload da pasta `futturu-website-calculator` para `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Acesse **Configurações > Calculadora Futturu** para configurar
4. Use o shortcode `[futturu_calc]` em qualquer página

= Instalação via FTP =

1. Baixe o arquivo ZIP do plugin
2. Extraia o conteúdo
3. Faça upload da pasta extraída para `/wp-content/plugins/`
4. Ative o plugin no menu 'Plugins'

== Frequently Asked Questions ==

= Posso alterar os valores dos serviços? =

Sim! Acesse **Configurações > Calculadora Futturu** e altere todos os valores conforme necessário. As mudanças são aplicadas imediatamente.

= O cálculo é feito em tempo real? =

Sim, o valor é atualizado automaticamente sempre que o usuário altera alguma opção, sem necessidade de recarregar a página.

= Os dados são enviados por e-mail? =

Sim, ao preencher o formulário final, todos os dados (opções escolhidas + informações de contato) são enviados por e-mail para o endereço configurado (padrão: suporte@futturu.com.br).

= Posso personalizar as cores? =

Com certeza! O plugin inclui uma seção completa de personalização visual onde você pode definir cores primárias, secundárias, de destaque, fundo, texto, além de escolher entre 8 fontes profissionais.

= Funciona com meu tema WordPress? =

Sim! O plugin usa CSS isolado com variáveis CSS e é totalmente compatível com qualquer tema WordPress.

= É responsivo? =

Sim, a calculadora é totalmente responsiva e funciona perfeitamente em dispositivos móveis, tablets e desktops.
